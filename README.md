# nuxt-electron-example

## Build Setup

```bash
# install dependencies
$ yarn install

# .env
$ cp src/.env.example src/.env

# serve with hot reload
$ make run

# serve only nuxt
$ make nuxt

# build for staging and production
$ make build-stg-mac
$ make build-prd-mac
$ make build-stg-win
$ make build-prd-win

# clean build files
$ make clean

# jest test
$ make test

# linter
$ make lint
```
